About This Project

Welcome to the Student Performance Tracker!

This Django-based system allows students to analyze their academic performance dynamically using data from an Excel sheet. The features include:

  • 📊 View complete student data from the dataset
  • 🚨 Detect slow learners based on percentage
  • 📉 Identify weak subjects and visualize them using pie charts
  • 📚 Provide explanations and examples for difficult topics
  • ⏱ Countdown preparation timer with email reminders
  • 📝 Conduct quizzes based on the weak topic
  • 📧 Notify parents via email if performance doesn’t improve

This application is designed to provide academic support and ensure student progress with intelligent suggestions.
